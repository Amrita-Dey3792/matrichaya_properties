Image Guidelines

Logo Images

  • • Square format (1:1 ratio)
  • • Recommended: 200x200px
  • • PNG format preferred
  • • Transparent background

Banner Images

  • • Wide format (16:9 ratio)
  • • Recommended: 1200x675px
  • • JPG format preferred
  • • High contrast for text

Background Images

  • • Any format works
  • • Recommended: 1920x1080px
  • • Subtle patterns preferred
  • • Low opacity overlay
